Foxit PDF Reader opens the same document in under a second, installs in 78 MB, and stays out of your way. For everyday document work — reading, annotating, filling forms, signing — it covers everything Acrobat does for zero cost.

## Digital Signatures

Foxit supports three signature types:

**1. Drawn / Typed signature** — create a visual signature image, drag to any field  
**2. Certificate-based signature** (PKI) — sign with your local or smart-card certificate  
**3. Cloud signature** — integrate with DocuSign, Adobe Sign, AdobeSign via API

All signature types comply with PDF/A and ISO 32000-2 standards.

## Installation

```
1. Download the installer (78 MB)
2. Run FoxitPDFReader_setup.exe
3. Choose installation type:
   - Standard (recommended) — all features
   - Minimal — viewer only, no editing tools
4. Uncheck optional toolbar browser extensions if not needed
5. Launch Foxit from Start menu or taskbar pin
```

> **Tip:** During installation, uncheck "Set Foxit as default browser PDF viewer" if you prefer to keep that separate from file-association defaults.
