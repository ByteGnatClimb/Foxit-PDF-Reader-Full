# Foxit PDF Reader — Changelog

## 2025.2 — November 2025
- Chromium PDF rendering engine updated for better complex PDF support
- OCR: Arabic and Hebrew language packs added
- Document comparison: paragraph-level diff highlighting

## 2025.1 — August 2025
- Digital signature: cloud signature providers (DocuSign, AdobeSign) integration
- Tab management: restore closed tabs session feature
- Performance: 15% faster large document rendering (200+ pages)

## 2024.4 — May 2025
- PDF/A-3 compliance validation added
- Measurement tools: Area and Perimeter with real-world scale calibration
- Dark mode: improved contrast ratios for annotation colors

## 2024.3 — February 2025
- PDF to Word export: table structure preservation improved
- Stamp tool: custom image stamps from PNG/JPG
- Form filling: auto-save on field change

## 2024.2 — October 2024
- HTTP/3 PDF loading (direct URL open from web faster)
- Accessibility: improved screen reader support (NVDA/JAWS)

## 2024.1 — June 2024
- Initial 2024 series release
- Base install size reduced to 78 MB
- Startup time optimized to sub-1-second on modern SSDs
